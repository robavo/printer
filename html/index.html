Printer Service

POST JSON to /print to print a label.

Endpoint

POST /print
Content-Type: application/json

asset

Prints to Brother P-900W (36mm yellow tape)

FieldRequiredDescription
typeyes"asset"
printeryesPrinter IP address
serialyesAsset serial number
nameyesAsset name
dateyesDate string
{
  "type": "asset",
  "printer": "10.10.0.5",
  "serial": "SB-0001",
  "name": "Robot Arm A",
  "date": "2026-03-30"
}

sku

Prints to Brother QL-810W (38mm continuous tape)

FieldRequiredDescription
typeyes"sku"
printeryesPrinter IP address
skuyesSKU code
nameyesProduct name
{
  "type": "sku",
  "printer": "10.10.0.4",
  "sku": "WH-100",
  "name": "Widget Handle"
}

kit

Prints to Brother QL-810W (38mm continuous tape)

FieldRequiredDescription
typeyes"kit"
printeryesPrinter IP address
skuyesSKU code
nameyesKit name
dateyesAssembly date
whoyesAssembled by
assemblynoIf true, adds assembled checkbox + name line
{
  "type": "kit",
  "printer": "10.10.0.4",
  "sku": "KT-200",
  "name": "Assembly Kit",
  "date": "2026-03-30",
  "who": "Rob",
  "assembly": true
}

curl examples

curl -X POST http://localhost:3000/print \
  -H 'Content-Type: application/json' \
  -d '{"type":"asset","printer":"10.10.0.5","serial":"SB-0001","name":"Robot Arm A","date":"2026-03-30"}'

curl -X POST http://localhost:3000/print \
  -H 'Content-Type: application/json' \
  -d '{"type":"sku","printer":"10.10.0.4","sku":"WH-100","name":"Widget Handle"}'

curl -X POST http://localhost:3000/print \
  -H 'Content-Type: application/json' \
  -d '{"type":"kit","printer":"10.10.0.4","sku":"KT-200","name":"Assembly Kit","date":"2026-03-30","who":"Rob","assembly":true}'

Set PRINTER_QL and PRINTER_PT env vars to print sample labels on startup.
